A Generative Adversarial Network refers to two neural networks that â€œcompeteâ€ with each other, progressively improving the output toward a desired end goal. One network tries to generate novel output based on a training set (for example, it might try to create a new image of dog based on a dataset of puppy photos). The other network then tries to identify when that output is too different from the original training set (e.g. the first network draws a house instead of a dog).
